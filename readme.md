## 介绍

这是一个chrome浏览器插件，可以帮助你将网页复制的文字转变为markdown格式。

## 使用说明

1. 首先将本项目clone到你的本地主机
2. 在chrome中选择加载已解压的扩展程序，即可启用插件
3. 在网页中选择你希望转换的文字，点击右键，选择 copy as md即可
